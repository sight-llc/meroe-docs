---
title: Getting Started
description: Register, create your first customer, and receive a reconciled payment — in about 15 minutes.
---

This walks through the full happy path: register as a developer, create a customer with a dedicated NUBAN, and watch an inbound payment get automatically reconciled.

<Steps>
  <Step title="Sign up and get your API key">
    Create your developer account at https://meroe-developer-dashboard.vercel.app. After signing in, go to Apps → create an app, then API Keys → mint a key. Your secret key (`mr_test_sk_...`) is shown once — copy it now. See [Authentication](/docs/guides/authentication) for scopes and key rotation.
  </Step>

  <Step title="Create your first customer">
    Every customer gets a dedicated Nigerian bank account number (NUBAN), provisioned through Nomba and tied to a customer identity that Meroe tracks forever.

    ```bash
    curl -X POST https://meroe.ddns.net/v1/customers \
      -H "Authorization: Bearer mr_test_sk_..." \
      -H "Content-Type: application/json" \
      -d '{
        "fullName": "Aisha Mohammed",
        "kycTier": "TIER_1",
        "bvn": "12345678901"
      }'
    ```

    The response includes the customer profile and the account's `accountNumber`.
  </Step>

  <Step title="Trigger a test payment">
    In sandbox, send a real transfer from any bank app to your customer's NUBAN using Nomba's test sandbox, or use `POST /v1/webhook-subscriptions/{id}/test` to send a test event to your registered webhook URL. The [Sandbox](/docs/guides/sandbox) guide covers both approaches.
  </Step>

  <Step title="Watch it reconcile">
    Within seconds, the payment is matched and posted to the customer's balance. Confirm it:

    ```bash
    curl https://meroe.ddns.net/v1/customers/{id}/balance \
      -H "Authorization: Bearer mr_test_sk_..."
    ```

    The endpoint path `https://meroe.ddns.net/v1/customers/{id}/balance` returns the current balance. Read [Reconciliation](/docs/concepts/reconciliation) and [The Ledger](/docs/concepts/ledger) to understand the underlying mechanics.
  </Step>
</Steps>
